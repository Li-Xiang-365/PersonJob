# 爱尚学 - 智能教学管理系统

## 项目简介

**爱尚学** 是一款智能教学管理系统，采用 **小程序 + 后台管理** 双端架构。系统面向教育场景，为教师和学生提供课堂管理、作业发布与提交、签到、测验投票、公告通知等一站式教学互动功能。

## 功能特性

### 教师端（Web 管理后台）

- 📊 **数据看板** — 用户总数、课堂数量、测验/作业/公告/签到的统计概览
- 📚 **课堂管理** — 创建课堂、查看详情、编辑/删除课堂
- 📝 **作业管理** — 发布作业、按课堂/教师/状态筛选、查看/删除作业
- 📢 **公告管理** — 发布课堂公告
- ✅ **签到管理** — 发起签到活动
- 🗳️ **测验投票** — 创建测验和投票
- 💬 **弹幕管理** — 管理课堂弹幕/提问
- 👥 **用户管理** — 管理学生和教师账号
- ⚙️ **个人中心** — 修改个人信息

### 学生端（微信小程序）

- 🏠 **首页** — 快捷入口（我的课堂、加入课堂）、学习统计、学习小贴士
- 📖 **课堂详情** — 课堂公告、课堂作业、签到记录、测验投票、弹幕互动、退出课堂
- 📋 **作业列表** — 查看并提交作业
- 📢 **公告列表** — 浏览课堂公告
- ✅ **签到列表** — 参与课堂签到
- 🧪 **测验列表** — 参与课堂测验/投票
- 👤 **我的** — 个人信息、修改资料/密码、退出登录

## 技术架构

```
┌─────────────────────────────────────────────┐
│              学生端 (微信小程序)              │
│          UniApp + Vue3 + Pinia + SCSS       │
└──────────────────┬──────────────────────────┘
                   │ HTTP API
┌──────────────────▼──────────────────────────┐
│            后端服务 (Node.js)                │
│      Express + MySQL2 + CORS + Morgan       │
└──────────────────┬──────────────────────────┘
                   │
┌──────────────────▼──────────────────────────┐
│         教师端 (Web 管理后台)                │
│    Vue3 + TypeScript + Element Plus + Vite  │
└─────────────────────────────────────────────┘
```
